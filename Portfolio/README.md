# Premium Personal Portfolio

This project is a complete responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features
- Fixed animated navigation with mobile hamburger menu
- Hero section with typing animation
- Skills cards with hover effects
- Achievements timeline
- Responsive gallery with lightbox
- Resume section with download/view actions
- Smooth scroll reveal animations and floating background shapes

## Structure
- index.html — Page structure and content
- style.css — Complete visual design and responsive layout
- script.js — Interactivity, typing effect, navigation, reveal animations, and lightbox
- assets/ — Images and resume PDF assets

## Run locally
Open index.html in your browser, or serve the folder with a local static server such as:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000/
